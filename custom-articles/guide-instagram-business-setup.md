# מדריך: הגדרת חשבון אינסטגרם עסקי לאוטומציות

## למה צריך חשבון עסקי באינסטגרם?

בדיוק כמו פייסבוק, גם אינסטגרם מאפשר אוטומציות **רק לחשבונות עסקיים** (Instagram Business או Creator). חשבון אישי רגיל לא תומך ב-API ולא ניתן לחבר אותו ל-Make.com או ManyChat.

## מה אפשר לעשות עם חשבון עסקי?

- **קבלת התראות על תגובות** — דרך Webhook
- **שליחת DM אוטומטית** — דרך Instagram Messaging API
- **פרסום אוטומטי** — תמונות וסרטונים דרך API
- **חיבור ל-ManyChat** — בוטים שעונים ל-DM
- **Lead generation** — שילוב עם Facebook Lead Ads
- **אנליטיקס** — נתוני ביצועים דרך API

## שלב 1: המרה לחשבון עסקי

1. פתחו את אפליקציית אינסטגרם
2. כנסו ל-**פרופיל → ☰ → הגדרות → חשבון**
3. גללו למטה → **"עברו לחשבון מקצועי"**
4. בחרו **"עסק"** (Business) — לא Creator
5. בחרו קטגוריה (למשל: "שירותים עסקיים")
6. אשרו

## שלב 2: חיבור לדף פייסבוק (חובה!)

**זה השלב הכי חשוב** — בלי חיבור לדף פייסבוק, אוטומציות לא יעבדו.

1. באינסטגרם: **הגדרות → חשבון → חשבונות מקושרים**
2. בחרו **Facebook** והתחברו
3. בחרו את **הדף העסקי** שרוצים לקשר
4. אם אין לכם דף — צרו אחד (ראו מדריך "פרופיל פרטי vs דף עסקי")

### למה חייבים דף פייסבוק?
- Instagram API עובד **דרך Facebook Graph API**
- כל הכלים (Make.com, ManyChat) מתחברים דרך Facebook
- ה-Token שמאפשר גישה נוצר דרך Facebook OAuth

## שלב 3: חיבור ל-Make.com

1. ב-Make.com: הוסיפו מודול **Instagram Business**
2. לחצו **Create a Connection**
3. תתבקשו להתחבר לפייסבוק (כן, פייסבוק — לא אינסטגרם)
4. בחרו את הדף העסקי המקושר לאינסטגרם
5. אשרו את כל ההרשאות: instagram_basic, instagram_manage_messages, pages_read_engagement

## שלב 4: חיבור ל-ManyChat

1. ב-ManyChat: **Settings → Instagram**
2. לחצו **Connect Instagram Account**
3. התחברו עם Facebook ובחרו את הדף המקושר
4. ManyChat יבדוק שהחשבון עסקי ומחובר לדף
5. אשרו

## מה עובד ומה לא

### עובד עם אוטומציות:
- ✅ תגובות על פוסטים/ריליס — אפשר לקבל התראות
- ✅ הודעות DM — אפשר לשלוח ולקבל
- ✅ פרסום תמונות/סרטונים — דרך Content Publishing API
- ✅ אנליטיקס — נתוני חשיפה, מעורבות
- ✅ Story mentions — כשמישהו מתייג אתכם בסטורי

### לא עובד:
- ❌ לייקים אוטומטיים — אסור ומפר תנאי שימוש
- ❌ מעקב אוטומטי (Auto-follow) — פייסבוק חוסמת
- ❌ פרסום סטוריז דרך API — לא נתמך (נכון ל-2024)
- ❌ תגובות אוטומטיות מחשבון אישי — רק עסקי

## טיפים חשובים
- **חשבון Creator vs Business**: שניהם תומכים ב-API, אבל Business מומלץ כי יש לו יותר אפשרויות
- **אל תחליפו בין אישי לעסקי בתדירות** — זה עלול לגרום לבעיות חיבור
- **ודאו שהדף העסקי בפייסבוק מפורסם** — דף בטיוטה חוסם את חיבור האינסטגרם
- **אם החיבור ל-Make נשבר** — בדרך כלל הבעיה בצד הפייסבוק, לא האינסטגרם

## שגיאות נפוצות
- **"Instagram account is not a Business account"** — המירו לחשבון עסקי (שלב 1)
- **"No Facebook Page linked"** — חברו את האינסטגרם לדף פייסבוק (שלב 2)
- **"Permissions error"** — חדשו את החיבור ב-Make.com ואשרו את כל ההרשאות
- **"Media not found"** — הפוסט נמחק או שאין גישה. ודאו שהחשבון ציבורי (לא פרטי)
- **"Rate limit reached"** — יותר מדי בקשות. המתינו שעה ונסו שוב
