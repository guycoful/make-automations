# מתכון אוטומציה: ליד מטופס פייסבוק → הודעת WhatsApp + CRM

## תיאור התהליך
אוטומציה שתופסת ליד חדש מטופס לידים ממומן בפייסבוק (Facebook Lead Ad), שולחת לו הודעת WhatsApp מיידית עם הודעת ברוכים הבאים, ומכניסה את הפרטים ל-CRM או לטבלת Google Sheets.

## מודולים נדרשים ב-Make.com
- **Facebook Lead Ads — Watch Leads**: טריגר שתופס לידים חדשים מקמפיינים
- **HTTP — Make a Request**: שליחת הודעת WhatsApp דרך GreenAPI
- **Google Sheets — Add a Row** (או Airtable/Monday): שמירת פרטי הליד

## חיבורים מוקדמים
1. חיבור Facebook Lead Ads: הגדרות → Connections → Facebook Lead Ads → חשבון פייסבוק עם גישה לדף העסקי ולמודעות
2. חיבור GreenAPI: Instance ID + API Token מהדשבורד של GreenAPI
3. חיבור Google Sheets: חשבון Google עם גישה לגיליון הרלוונטי

## שלבים לבניית הסנריו

### שלב 1: טריגר — Facebook Lead Ads
1. צרו סנריו חדש ב-Make.com
2. הוסיפו מודול **Facebook Lead Ads → Watch Leads**
3. בחרו את החיבור לפייסבוק
4. בחרו את הדף העסקי ואת טופס הלידים הרלוונטי
5. הגדירו From: תאריך התחלה לקליטה

### שלב 2: שליחת הודעת WhatsApp
1. הוסיפו מודול **HTTP → Make a request**
2. הגדירו:
   - URL: `https://api.green-api.com/waInstance{{InstanceID}}/sendMessage/{{APIToken}}`
   - Method: POST
   - Body type: JSON
   - Body:
     ```json
     {
       "chatId": "{{phone_number}}@c.us",
       "message": "שלום {{full_name}}! 👋\n\nתודה שהשארת פרטים.\nנציג שלנו יחזור אליך בהקדם.\n\nבינתיים, אפשר לשאול אותי כל שאלה כאן!"
     }
     ```
3. שימו לב: מספר הטלפון מגיע מטופס הליד. צריך לנקות אותו לפורמט 972XXXXXXXXX
4. אפשר להשתמש בפונקציות טקסט של Make: `replace(phone; "+"; "")` ו-`replace(result; "0"; "972"; 1)`

### שלב 3: שמירה ב-Google Sheets (CRM פשוט)
1. הוסיפו מודול **Google Sheets → Add a Row**
2. בחרו את הגיליון המתאים
3. מפו את השדות:
   - שם מלא ← `{{full_name}}`
   - טלפון ← `{{phone_number}}`
   - אימייל ← `{{email}}`
   - תאריך ← `{{now}}`
   - סטטוס ← "ליד חדש"
   - מקור ← "Facebook Lead Ad"

### שלב 4: הפעלה ובדיקה
1. שמרו את הסנריו
2. לחצו Run Once
3. שלחו ליד מבדיקה דרך טופס הלידים בפייסבוק (כפתור Preview)
4. ודאו שהודעת WhatsApp נשלחה ושהשורה נוספה לגיליון
5. הפעילו את הסנריו ותזמנו בדיקה כל 5 דקות

## חלופה: שימוש ב-Monday.com במקום Sheets
במקום Google Sheets, אפשר להשתמש במודול **Monday.com → Create Item**:
1. בחרו את הבורד שמנהל את הלידים
2. מפו את העמודות: שם, טלפון, אימייל, סטטוס, מקור

## חלופה: WhatsApp דרך ManyChat
אם אתם משתמשים ב-ManyChat עם WhatsApp:
1. צרו Flow חדש ב-ManyChat עם הודעת ברוכים הבאים
2. ב-Make.com: הוסיפו מודול ManyChat → Send Content במקום GreenAPI
3. זה מאפשר גם ליצור שיחה אינטראקטיבית (בוט) ולא רק הודעה חד-כיוונית

## טיפים חשובים
- ליד מפייסבוק מגיע עם השדות שהגדרתם בטופס — ודאו שטופס הלידים כולל שדה טלפון
- מומלץ לשלוח את ההודעה תוך דקות ספורות מרגע ההרשמה — לידים חמים מגיבים טוב יותר
- הוסיפו Router אם רוצים לשלוח הודעות שונות לפי סוג הקמפיין
- אל תשכחו Error Handler — אם מספר הטלפון שגוי, הסנריו לא צריך ליפול

## שגיאות נפוצות
- **ליד לא נתפס**: ודאו שהטופס מחובר לדף הנכון ושהחיבור לפייסבוק פעיל
- **מספר טלפון לא תקין**: השתמשו בפונקציות Replace לנקות את המספר מתווים מיוחדים
- **שגיאת GreenAPI 466**: המספר לא רשום ב-WhatsApp — הוסיפו Error Handler מסוג Ignore
- **כפילויות ב-Sheets**: הוסיפו מודול Search Row לפני Add Row כדי לבדוק אם הליד כבר קיים
