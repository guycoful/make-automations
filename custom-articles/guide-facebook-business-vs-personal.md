# מדריך: פרופיל פרטי vs דף עסקי בפייסבוק — מה צריך לאוטומציות?

## למה פרופיל פרטי לא עובד עם אוטומציות?

פייסבוק מפרידה בצורה ברורה בין **פרופיל אישי** (Personal Profile) לבין **דף עסקי** (Facebook Page). כל כלי האוטומציה — Make.com, ManyChat, Zapier ואחרים — עובדים **רק עם דפים עסקיים**, ולא עם פרופילים פרטיים.

### הסיבות:
1. **API של פייסבוק** — Facebook Graph API לא חושף נתונים של פרופילים פרטיים לאפליקציות חיצוניות
2. **הגנת פרטיות** — פייסבוק חוסמת גישה לפרופילים פרטיים כדי להגן על המשתמשים
3. **תנאי שימוש** — שימוש באוטומציות על פרופיל פרטי מפר את תנאי השירות וגורם לחסימת חשבון
4. **Webhooks** — רק דפים עסקיים תומכים ב-Webhooks שמאפשרים לקבל התראות בזמן אמת

## מה אפשר לעשות עם דף עסקי?

- **קבלת התראות על תגובות** — כשמישהו מגיב לפוסט בדף
- **מענה אוטומטי להודעות** — דרך Messenger API
- **Lead Ads** — טפסי לידים ישירות מפייסבוק
- **פרסום אוטומטי** — שליחת פוסטים דרך API
- **חיבור ל-ManyChat** — בוטים למסנג'ר
- **חיבור ל-Make.com** — כל הטריגרים והאקשנים של פייסבוק

## איך להקים דף עסקי?

### שלב 1: יצירת הדף
1. היכנסו לפייסבוק עם הפרופיל האישי שלכם
2. לחצו על **"צור" → "דף"** (או כנסו ל-facebook.com/pages/create)
3. בחרו שם לדף (שם העסק)
4. הוסיפו קטגוריה (למשל: "שירותים עסקיים", "חנות מקוונת")
5. העלו תמונת פרופיל ותמונת כיסוי

### שלב 2: הגדרות בסיסיות
1. מלאו את כל פרטי הדף: כתובת, טלפון, שעות פעילות
2. הוסיפו כפתור CTA (למשל: "צרו קשר", "הזמינו עכשיו")
3. פרסמו לפחות 2-3 פוסטים ראשונים

### שלב 3: הגדרת Business Manager (מומלץ מאוד)
1. כנסו ל-business.facebook.com
2. צרו חשבון Business Manager
3. הוסיפו את הדף לחשבון
4. זה נדרש עבור: Lead Ads, Facebook Login, ו-API גישה מתקדמת

## חיבור הדף לכלי אוטומציה

### חיבור ל-Make.com
1. ב-Make, הוסיפו מודול Facebook Pages
2. לחצו Create a Connection
3. תתבקשו להתחבר עם הפרופיל האישי שלכם
4. בחרו את הדפים שאתם רוצים לתת גישה אליהם
5. אשרו את כל ההרשאות: manage_pages, pages_messaging, leads_retrieval

### חיבור ל-ManyChat
1. הירשמו ל-ManyChat עם חשבון הפייסבוק שלכם
2. בחרו את הדף העסקי
3. ManyChat יבקש הרשאות — אשרו הכל
4. הדף חייב להיות **מפורסם** (לא טיוטה)

## שאלות נפוצות

### "יש לי קבוצת פייסבוק — אפשר לחבר אותה?"
לא ישירות. קבוצות פייסבוק לא תומכות ברוב ה-APIs. **הפתרון**: צרו דף עסקי, קשרו אותו לקבוצה, והשתמשו בדף לאוטומציות.

### "מה אם אין לי עסק רשום?"
לא צריך עסק רשום! כל אחד יכול ליצור דף עסקי בפייסבוק. תוכלו לקרוא לו בשם המותג, הפרויקט או השירות שלכם.

### "האם אני צריך לשלם?"
לא. יצירת דף עסקי היא חינמית. התשלום הוא רק אם תרצו לפרסם מודעות ממומנות.

### "מה עם Instagram?"
גם אינסטגרם דורש **חשבון עסקי** (Instagram Business) מחובר לדף פייסבוק. בלי זה — אי אפשר להשתמש ב-API ובאוטומציות.

## טיפים חשובים
- תמיד חברו את הדף ל-Business Manager — זה פותח אפשרויות מתקדמות
- אל תשתמשו בפרופיל פרטי לשליחת הודעות אוטומטיות — פייסבוק תחסום אתכם
- שמרו את ה-Page Access Token — הוא נדרש לכל חיבורי ה-API
- אם הדף חדש, תנו לו כמה ימים לפני שמנסים חיבורי API מתקדמים

## שגיאות נפוצות
- **"This page doesn't have permission"** — ודאו שנתתם את כל ההרשאות בעת חיבור האפליקציה
- **"Page not found"** — הדף צריך להיות מפורסם (Published), לא בטיוטה
- **"Invalid OAuth token"** — Token פג תוקף, חדשו את החיבור ב-Make.com
- **"You don't have admin access"** — רק מנהלי הדף יכולים לחבר אוטומציות
