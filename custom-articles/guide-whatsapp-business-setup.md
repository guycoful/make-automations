# מדריך: הגדרת WhatsApp Business לאוטומציות

## מה צריך כדי לשלוח הודעות WhatsApp אוטומטיות?

כדי לשלוח הודעות WhatsApp דרך Make.com או כלי אוטומציה אחר, יש שלוש אפשרויות עיקריות:

1. **GreenAPI** — פתרון פשוט ומהיר, מתאים לעסקים קטנים
2. **WhatsApp Business API הרשמי** — דרך Meta, לעסקים גדולים
3. **ספקי צד שלישי** — Twilio, MessageBird, WATI ועוד

## אפשרות 1: GreenAPI (המומלצת למתחילים)

### למה GreenAPI?
- **פשוט להתקנה** — תוך 5 דקות
- **לא דורש אישור Meta** — עובד מיד
- **מחיר נגיש** — תוכניות מ-$5/חודש
- **עובד מצוין עם Make.com** — דרך HTTP module

### שלב 1: הרשמה ל-GreenAPI
1. כנסו ל-green-api.com
2. הירשמו וצרו חשבון
3. קבלו Instance ID ו-API Token

### שלב 2: חיבור WhatsApp
1. בממשק של GreenAPI, לחצו "Connect WhatsApp"
2. יופיע QR Code
3. פתחו WhatsApp בטלפון → הגדרות → מכשירים מקושרים → קשר מכשיר
4. סרקו את ה-QR Code
5. המתינו לאישור — הסטטוס ישתנה ל-"Connected"

### שלב 3: בדיקה
שלחו הודעת בדיקה דרך ה-API:
```json
{
  "chatId": "972501234567@c.us",
  "message": "בדיקה — ההודעה הגיעה!"
}
```

### שלב 4: חיבור ל-Make.com
1. הוסיפו מודול **HTTP → Make a Request**
2. URL: `https://api.green-api.com/waInstance{{InstanceID}}/sendMessage/{{APIToken}}`
3. Method: POST
4. Body type: JSON
5. הזינו את ה-Body עם chatId ו-message

## אפשרות 2: WhatsApp Business API הרשמי

### למי מתאים?
- עסקים שצריכים לשלוח **יותר מ-1000 הודעות ביום**
- עסקים שרוצים **שליחת Template Messages** (הודעות מאושרות מראש)
- חברות שדורשות **תאימות רגולטורית מלאה**

### תהליך ההרשמה
1. צרו חשבון Meta Business (business.facebook.com)
2. אמתו את העסק (צריך מסמכים)
3. הגדירו WhatsApp Business Account
4. הגדירו מספר טלפון ייעודי (לא יכול להיות באותו מספר של WhatsApp רגיל)
5. צרו Template Messages ושלחו לאישור Meta

### חסרונות
- **תהליך אישור ארוך** — בין יום לשבוע
- **דורש מספר טלפון ייעודי** — לא יכול להיות אותו מספר של ה-WhatsApp הפרטי
- **Template Messages בלבד** — לא ניתן לשלוח הודעות חופשיות (אלא אם הלקוח יצר קשר קודם)
- **יקר יותר** — תשלום לפי הודעה

## אפשרות 3: ספקי צד שלישי

### Twilio
- **יתרון**: אמין מאוד, API מעולה, תמיכה טכנית
- **חיסרון**: יקר ($0.005-0.05 להודעה), הגדרה מורכבת
- **מתאים ל**: עסקים בינוניים-גדולים

### WATI
- **יתרון**: ממשק בעברית, תמיכה ישראלית
- **חיסרון**: מחיר גבוה יחסית
- **מתאים ל**: עסקים ישראליים שרוצים תמיכה מקומית

## פורמט מספרי טלפון

כל ה-APIs דורשים מספר בפורמט בינלאומי:

- **נכון**: `972501234567` (בלי + בהתחלה)
- **לא נכון**: `0501234567`, `+972-50-123-4567`

### נוסחת ניקוי ב-Make.com
```
replace(replace(replace(phone; "-"; ""); " "; ""); "+"; "")
```

### המרה מ-0 ל-972
```
if(substring(cleanPhone; 0; 1) = "0"; "972" + substring(cleanPhone; 1); cleanPhone)
```

## טיפים חשובים
- **אל תשלחו ספאם** — WhatsApp חוסם מספרים ששולחים הודעות לא רצויות
- **בקשו הסכמה** — תמיד ודאו שהלקוח הסכים לקבל הודעות
- **הגבילו תדירות** — מומלץ לא יותר מ-2-3 הודעות ביום ללקוח
- **הוסיפו אפשרות ביטול** — "שלח 'הפסק' כדי להפסיק לקבל הודעות"
- **שמרו את ה-QR Code session** — אם מתנתקים, צריך לסרוק שוב

## שגיאות נפוצות
- **"Phone number is not registered"** — המספר לא רשום ב-WhatsApp, ודאו שהמספר נכון
- **"QR Code expired"** — הסנו מחדש את ה-QR Code ב-GreenAPI
- **"Instance not authorized"** — ה-WhatsApp התנתק, צריך לסרוק QR שוב
- **"Message not sent — blocked"** — הנמען חסם את המספר שלכם
- **"Rate limit exceeded"** — שלחתם יותר מדי הודעות, המתינו ונסו שוב
