# מתכון אוטומציה: מילוי טופס גוגל → WhatsApp + Google Sheets

## תיאור התהליך
אוטומציה שמזהה מילוי טופס Google Forms, שומרת את הנתונים ב-Google Sheets, ושולחת הודעת WhatsApp אוטומטית לממלא הטופס עם אישור קבלה.

## מודולים נדרשים ב-Make.com
- **Google Forms — Watch Responses**: טריגר שמאזין לתשובות חדשות
- **Google Sheets — Add a Row**: שמירת הנתונים בגיליון
- **HTTP — Make a Request**: שליחת WhatsApp דרך GreenAPI

## חיבורים מוקדמים
1. חיבור Google ל-Make: OAuth עם חשבון Google שמכיל את הטופס
2. חיבור GreenAPI: Instance ID + API Token

## שלבים לבניית הסנריו

### שלב 1: טריגר — Google Forms
1. צרו סנריו חדש ב-Make.com
2. הוסיפו מודול **Google Forms → Watch Responses**
3. בחרו את החיבור לגוגל
4. בחרו את הטופס הרלוונטי
5. Make יציג את כל שדות הטופס כמשתנים זמינים

### שלב 2: שמירה ב-Google Sheets
1. הוסיפו מודול **Google Sheets → Add a Row**
2. בחרו את הגיליון (אפשר ליצור אחד ייעודי או להשתמש בגיליון התשובות שנוצר אוטומטית)
3. מפו את השדות: שם, טלפון, אימייל, תאריך הגשה, וכל שדה נוסף מהטופס

### שלב 3: ניקוי מספר טלפון
1. הוסיפו מודול **Tools → Set Variable** (אופציונלי אבל מומלץ)
2. השתמשו בפונקציות:
   - `replace(phone; "-"; "")`
   - `replace(result; " "; "")`
   - `replace(result; "+"; "")`
   - אם מתחיל ב-0: `replace(result; "0"; "972"; 1)`

### שלב 4: שליחת WhatsApp
1. הוסיפו מודול **HTTP → Make a request**
2. הגדירו:
   - URL: `https://api.green-api.com/waInstance{{InstanceID}}/sendMessage/{{APIToken}}`
   - Method: POST
   - Body type: JSON
   - Body:
     ```json
     {
       "chatId": "{{cleanPhone}}@c.us",
       "message": "שלום {{name}}! ✅\n\nקיבלנו את הטופס שלך.\nניצור איתך קשר בהקדם.\n\nתודה!"
     }
     ```

### שלב 5: הפעלה ובדיקה
1. שמרו את הסנריו
2. הפעילו Run Once
3. מלאו את הטופס בעצמכם לבדיקה
4. ודאו: שורה חדשה ב-Sheets + הודעת WhatsApp

## וריאציה: שליחה גם למנהל
אם רוצים שגם המנהל יקבל התראה:
1. הוסיפו **Router** אחרי הטריגר
2. נתיב 1: WhatsApp ללקוח (כמו למעלה)
3. נתיב 2: WhatsApp למנהל עם כל פרטי הליד
4. נתיב 3: Google Sheets

## טיפים חשובים
- ודאו שהטופס כולל שדה טלפון מסוג Short Answer (לא מספר — כדי לתמוך בפורמטים שונים)
- Google Forms לא שולח אירוע בזמן אמת — Make בודק כל X דקות (תזמון). הגדירו תזמון של כל 5 דקות לתגובה מהירה
- אפשר להוסיף תנאי לפי תשובות: לדוגמה, הודעה שונה למי שבחר "חבילה פרימיום" לעומת "חבילה בסיסית"

## שגיאות נפוצות
- **תשובות לא נקלטות**: ודאו שהטופס ששייכתם הוא הנכון ושהחיבור לגוגל תקף
- **מספר טלפון שגוי**: תמיד נקו את המספר מתווים מיוחדים לפני שליחה
- **הודעה כפולה**: הגדירו את הטריגר ל-Limit 1 והשתמשו ב-Deduplication
