# מתכון אוטומציה: תגובה בפוסט פייסבוק → הודעת WhatsApp

## תיאור התהליך
אוטומציה שמזהה תגובה חדשה על פוסט בדף הפייסבוק העסקי שלך ושולחת הודעת WhatsApp אוטומטית — לך כהתראה, או ישירות למגיב עם תשובה מותאמת.

## מודולים נדרשים ב-Make.com
- **Facebook Pages — Watch Comments**: טריגר שמאזין לתגובות חדשות על פוסטים בדף העסקי
- **WhatsApp Business (GreenAPI) — Send Message**: שליחת הודעת WhatsApp

## חיבורים מוקדמים
1. חיבור Facebook ל-Make: הגדרות → Connections → Facebook Pages → חיבור עם חשבון פייסבוק שמנהל את הדף העסקי
2. חיבור GreenAPI ל-Make: הגדרות → Connections → HTTP → הזנת Instance ID ו-API Token מחשבון GreenAPI

## שלבים לבניית הסנריו

### שלב 1: הוספת טריגר Facebook
1. צרו סנריו חדש ב-Make.com
2. הוסיפו מודול **Facebook Pages → Watch Comments**
3. בחרו את החיבור לפייסבוק שהגדרתם
4. בחרו את הדף העסקי הרלוונטי
5. הגדירו מאיזה תאריך לצפות בתגובות (Where to start)

### שלב 2: הוספת פילטר (אופציונלי)
אם רוצים לסנן רק תגובות מסוימות:
1. הוסיפו פילטר בין הטריגר למודול הבא
2. הגדירו תנאי, לדוגמה: Message contains "מחיר" או "פרטים"
3. כך רק תגובות רלוונטיות יפעילו את ההמשך

### שלב 3: שליחת הודעת WhatsApp דרך GreenAPI
1. הוסיפו מודול **HTTP → Make a request**
2. הגדירו:
   - URL: `https://api.green-api.com/waInstance{{InstanceID}}/sendMessage/{{APIToken}}`
   - Method: POST
   - Body type: JSON
   - Body:
     ```json
     {
       "chatId": "972XXXXXXXXX@c.us",
       "message": "תגובה חדשה בפייסבוק!\n\nמגיב: {{name}}\nתוכן: {{message}}\nפוסט: {{post_id}}"
     }
     ```
3. במקום מספר הטלפון הקבוע, אפשר להשתמש במספר שלכם לקבלת התראות

### שלב 4: הפעלה ובדיקה
1. שמרו את הסנריו
2. לחצו Run Once לבדיקה
3. כתבו תגובה בפוסט בדף הפייסבוק שלכם
4. ודאו שההודעה הגיעה ב-WhatsApp
5. אם הכל עובד — הפעילו את הסנריו (ON) והגדירו תזמון

## חלופה: שימוש ב-ManyChat
אם יש לכם חשבון ManyChat מחובר לפייסבוק ול-WhatsApp:
1. צרו אוטומציה חדשה ב-ManyChat
2. בחרו טריגר: **Facebook Comments** → בחרו את הפוסט הרלוונטי
3. הוסיפו Action: **Send WhatsApp Message**
4. כתבו את תוכן ההודעה עם משתנים דינמיים כמו שם המגיב

## טיפים חשובים
- ודאו שהדף העסקי בפייסבוק מחובר נכון ושיש לכם הרשאות ניהול
- ב-GreenAPI צריך שהטלפון יהיה מחובר ופעיל כל הזמן
- מומלץ להוסיף Error Handler מסוג Ignore כדי שהסנריו לא ייעצר בגלל שגיאה בודדת
- אם אתם שולחים הודעות ללקוחות (לא רק לעצמכם), שימו לב לחוק הספאם ולקבלת הסכמה מראש

## שגיאות נפוצות
- **שגיאת 401 מ-GreenAPI**: בדקו שה-Instance ID וה-API Token נכונים
- **לא מגיעות תגובות**: ודאו שהטריגר מוגדר לדף הנכון ושהסנריו מופעל
- **הודעה לא נשלחת**: ודאו שמספר הטלפון בפורמט בינלאומי (972 בלי 0) ושחשבון ה-WhatsApp פעיל
